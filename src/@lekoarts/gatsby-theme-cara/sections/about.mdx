## About

    アニメ・ゲーム・書籍・ガジェットなど、世の中にはたくさんのおもしろいプロダクトがあります。そんなワクワクするようなプロダクトを１人のユーザーとして愛しています！

### 島津 岳明について

はじめまして、島津岳明（Shimatsu Takeaki）

1999 年 8 月 17 日生まれ

好きな言葉は、『_逃げたら１つ、進めば２つ_』

中学校でアニメにハマり、ソードアートオンラインの世界に憧れを抱く

最終的には VR/AR/MR の UI デザイン・アプリケーション開発に携わりたいと思っており、ひとまずプログラミングに慣れるために Web 制作から攻めている。

<br />

高校卒業後に Web 制作をはじめ、独学＆プログラミングスクール ZeroPlus で学習後、フリーランス Web デザイナーを始める。

１年後、ZeroPlus の運営元である株式会社 TOMAP でプログラミング講師やサービスの作成の管理、カリキュラムのコーディングルールの作成などを行う。

そして、2023 年現在、再び憧れに向けて、Web 制作・開発から再始動する。
